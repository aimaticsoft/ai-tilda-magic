

## План: добавить новые аналитические блоки в отчёт

Отчёт уже содержит много данных. Добавим метрики, которых не хватает для полноценного анализа.

### Новые блоки

1. **Качество источников трафика**
   - Для каждого источника показать среднее время и средний скролл
   - Позволяет понять, какой трафик качественный, а какой "мусорный"
   - Формат: `google.com — 12 визитов, ср. 2 мин 15 сек, скролл 67%`

2. **Средние клики на визит + CTR по устройствам**
   - Общее среднее количество кликов на визит
   - Разбивка engagement-метрик по устройствам (mobile vs desktop): время, скролл, клики
   - Помогает понять, удобен ли сайт на мобильных

3. **Типичный путь пользователя (User Journey)**
   - Топ-3 самых частых последовательности секций
   - Показывает, как люди реально проходят сайт
   - Формат: `О компании → Услуги → Калькулятор → Контакты (15 раз)`

4. **Точка потери (Drop-off)**
   - Последняя секция, которую видел пользователь перед уходом
   - Показывает, где люди теряют интерес
   - Формат: `FAQ — 23% уходят после этой секции`

5. **Расширенные инсайты**
   - Сравнение mobile vs desktop engagement
   - Лучший и худший источник трафика по качеству
   - Рекомендация по времени показа рекламы на основе пиковых часов
