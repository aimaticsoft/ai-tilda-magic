

## Генерация логотипов с помощью ИИ

### Что сделаем

Сгенерируем 8 настоящих логотипов-картинок через Lovable AI (модель `google/gemini-3-pro-image-preview`) и заменим SVG-иконки на реальные изображения.

### Процесс

1. **Создадим edge-функцию** `generate-logo` для генерации логотипов через AI
2. **Сгенерируем 8 логотипов** для каждой компании:
   - CleanPro (зелёный, капля/чистота)
   - TravelMarket (синий, глобус/путешествия)
   - СИБТЭ (оранжевый, энергетика)
   - NovaTech (фиолетовый, технологии)
   - BrightWave (голубой, волна/маркетинг)
   - Кристалл (розовый, ритейл)
   - SkyLine Pro (индиго, недвижимость)
   - Импульс (красный, финтех)

3. **Сохраним логотипы** в Storage bucket (создадим bucket `logos`)
4. **Обновим `TrustBadgesSection.tsx`** — заменим SVG-компоненты на `<img>` с URL из Storage

### Файлы
- `supabase/functions/generate-logo/index.ts` — новая edge-функция для генерации
- `src/components/TrustBadgesSection.tsx` — замена SVG на img
- Database migration: создание storage bucket `logos`

### Альтернативный подход (проще и надёжнее)
Вместо runtime-генерации, сгенерируем логотипы один раз через edge-функцию, загрузим в storage, и захардкодим URL в компоненте. Это быстрее и не требует генерации при каждой загрузке страницы.
