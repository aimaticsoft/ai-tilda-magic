### 2. Исчезающий текст на кнопках `btn-neon`

**Проблема:** CSS `::before` перекрывает текст, если текст — прямой текстовый узел (не обёрнут в `<span>`). Правило `btn-neon > *` ставит `z-index: 1` только для дочерних элементов, но не для текстовых узлов.

**Решение в `src/index.css`:** Добавить `z-index: 1` и `position: relative` прямо на `.btn-neon`, чтобы текст всегда был выше `::before`:

```css
.btn-neon::before {
  z-index: -1;  /* было 0, станет -1 */
}
```

И убрать правило `.btn-neon > *` — оно больше не нужно. Это исправит все кнопки разом.
