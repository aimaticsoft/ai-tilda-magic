

## План: Заменить клиентов на выдуманные компании с SVG-логотипами

### Что сделаем

Заменим текущие компании с эмодзи на 8 выдуманных брендов с уникальными SVG-логотипами (инлайн-иконки). Названия — микс русских и английских, стилизованные под реальные бренды.

### Список компаний

| Название | Отрасль | Логотип (SVG-иконка) |
|----------|---------|---------------------|
| NovaTech | IT | Абстрактная N из линий |
| Сфера Групп | Консалтинг | Круг с сегментами |
| BrightWave | Маркетинг | Волна |
| Кристалл | Ритейл | Ромб/кристалл |
| SkyLine Pro | Недвижимость | Силуэт здания |
| Вектор Плюс | Логистика | Стрелка вправо-вверх |
| GreenField | Агро | Лист |
| Импульс | Финтех | Молния/пульс |

### Техническая реализация

**Файл: `src/components/TrustBadgesSection.tsx`**
- Заменить массив `clients` — вместо `icon: "🧹"` будет `logo: () => <svg>...</svg>` — инлайн SVG-компоненты
- Каждый логотип — простая геометрическая фигура 28x28px в цвете `currentColor` (адаптируется к теме)
- Рендер: вместо `<span className="text-2xl">{client.icon}</span>` → `<client.logo />`

Один файл, никаких внешних зависимостей.
