

## План: расширить ежедневный отчёт полезной аналитикой

Сейчас отчёт содержит: количество визитов, среднее время, скролл, устройства, источники, секции и клики. Добавим метрики, которые реально помогают принимать решения.

### Новые блоки в отчёте

1. **Показатель вовлечённости (Engagement)**
   - Процент «вовлечённых» визитов (время > 60 сек И скролл > 50%) vs «отказы» (время < 15 сек)
   - Bounce rate — доля посетителей с временем < 15 секунд

2. **Воронка по секциям**
   - Показать % посетителей, дошедших до каждой ключевой секции (калькулятор, демо, контакты)
   - Формат: `📞 Контакты — 23% посетителей дошли`

3. **Пиковые часы**
   - Распределение визитов по часам (МСК), показать топ-3 самых активных часа
   - Помогает понять, когда запускать рекламу

4. **Сравнение с предыдущим периодом**
   - Для daily: сравнить со вчера (визиты, время, скролл)
   - Показать стрелки ↑↓ и процент изменения

5. **Глубина просмотра**
   - Распределение скролла по диапазонам: 0-25%, 25-50%, 50-75%, 75-100%
   - Показывает, сколько людей видят нижнюю часть сайта

### Что изменится

**Файл:** `supabase/functions/send-daily-stats/index.ts`
- Добавить запрос данных за предыдущий аналогичный период для сравнения
- Вычислить bounce rate, engagement rate, воронку секций, пиковые часы, распределение скролла
- Добавить новые блоки в текст сообщения между существующими разделами

Структура отчёта станет:
```text
📊 Ежедневный отчёт
📅 01.04.2026

━━━ ПОСЕТИТЕЛИ ━━━
(существующее + bounce rate + engagement rate + сравнение ↑↓)

━━━ ПИКОВЫЕ ЧАСЫ ━━━
(топ-3 часа по количеству визитов)

━━━ ГЛУБИНА ПРОСМОТРА ━━━
(распределение скролла по диапазонам)

━━━ ВОРОНКА СЕКЦИЙ ━━━
(% дошедших до калькулятора, демо, контактов)

━━━ СЕКЦИИ ━━━
(существующее)

━━━ КЛИКИ ━━━
(существующее)

💡 Вывод: (расширенные инсайты)
```
